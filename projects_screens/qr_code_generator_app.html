Customizable QR Code Generator App

This Flutter-based QR Code Generator App was a personal project built to strengthen my Flutter fundamentals and experiment with local storage and clean UI design.

The app allows users to create QR codes for URLs, label them with a custom icon for easy identification, and save them locally.

It’s a lightweight utility app with a focus on usability, organization, and clean interface design.

App description

QR Code Generation

Used the qr_flutter package to generate clean QR codes for user-input URLs.

Provided a simple and responsive form for entering and submitting URLs.

Local Storage & Organization

Implemented local saving using shared_preferences to store generated QR codes and their labels.

Allowed users to add icon tags next to labels to easily recognize their saved QR codes.

Simple & Clean UI

Designed and built a minimal, user-friendly interface for generating and browsing saved codes.

Focused on fast interaction and intuitive user flow.
